Customise styles for media library.

== Description ==
Custom style infuser for admin.
Opens under Settings > GrandChild Editor 
Provides a low overhead editor to override child theme and parent theme styles.

== Features ==
* Select priority level of enqueue
* Saves to database so you can keep intact when deactivated.
* Enable or disable sizer styles at any time.

== Installation ==

This section describes how to install the plugin and get it working.
1. Upload `grandchile` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
